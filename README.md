# Epstein Files Analysis Dashboard (Power BI)

## Overview

This project presents an **interactive Power BI dashboard** analyzing publicly available data related to the *Epstein files*. The goal of the dashboard is to explore patterns across **flight logs, document mentions, and relationships between individuals** using visual analytics.

The dashboard transforms raw datasets into an **investigative-style visualization interface**, allowing users to explore trends, connections, and timelines interactively.

This project was built as a **data visualization and storytelling exercise**, demonstrating how business intelligence tools can be used to analyze complex datasets.

---

## Dashboard Features

The dashboard includes multiple analytical sections designed to highlight different aspects of the dataset.

### Key Metrics

Summary indicators provide a quick overview of the dataset including:

- Total People Mentioned  
- Total Flights Recorded  
- Total Documents  
- Total Mentions Across Documents  

---

### Flight Activity Analysis

Visualizations highlight patterns in flight data including:

- Flights per year
- Geographic flight origins
- Most frequent individuals appearing in flights

These visuals help identify **temporal and geographical trends** in the dataset.

---

### Document Mentions

The dashboard analyzes document references across multiple document types such as:

- Court filings  
- Depositions  
- Contact book entries  
- Flight logs  
- Email records  
- Interview transcripts  

A **word cloud visualization** highlights the most frequently referenced document types.

---

### Relationship Network

A **connection map** visualizes relationships between key individuals appearing in the dataset. This network representation helps illustrate how different people are linked within the records.

---

### Timeline of Events

A timeline highlights major events connected to the case including:

- **2008** – Epstein Arrested  
- **2015** – Lawsuit Filed  
- **2019** – Epstein Death  
- **2021** – Maxwell Convicted  

This provides important **historical context** for the dataset.
